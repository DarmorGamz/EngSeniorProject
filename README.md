## Getting Started

<!-- REQUIREMENTS -->
### Requirements

- Python 3.11.9

### Installation

#### Clone this repository
```bash
git clone https://github.com/DarmorGamz/Tickermind.git
```

#### Build the Docker image
```bash
docker compose up --build
```
